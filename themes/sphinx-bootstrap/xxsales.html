Designed for the web

Vixo has been designed and built as a web-spreadsheet from the ground up, its API is powerful and direct and designed to make integrating a breeze.

Your API for your systems

The API is flexible and allows you to give systems partial and reduced permissions for better security and control.

Making integration easy

Spreadsheets are glueware, integrating information from people and systems is what they do - with Vixo you can build integrated dashboards in a trice.

The simplest API imaginable - but no simpler

Every object on a Vixo spreadsheet has a URL - you want to change a value just post to that URL - the simplest spreadsheet API on the web.

Putting your business users in control

Your business users can build the templates, heirarchies and aggregation they want in Vixo, leaving you to just put in the raw data - taking all the aggravation out of dashboard designing.
